Java语言概述

分类

  • java SE：标准版本
  • java EE：企业级版本
  • java ME：小型版（支持移动终端）

JDK、JRE和JVM

  1. JDK：Java开发工具包

    • JVM

    • 核心类库

    • 开发工具

  2. JRE：Java运行环境

    • JVM
    • 核心类库
    • 运行工具
  3. JVM：虚拟机：Java程序运行的地方
  • $JDK \supsetneq JRE \supsetneq JVM$

关键字与保留字

关键字：被Java赋予了特殊含义的英文单词

保留字：现有 Java 版本尚未使用，但以后版本可能会作为关键字使用。

  • 关键字的字母全部小写

关键字：

abstract assert boolean break byte
case catch char class const
continue default do double else
enum extends final finally float
for goto if implements import
instanceof int interface long native
new package private protected public
return strictfp short static super
switch synchronized this throw throws
transient try void volatile while

保留字：

byValue、cast、future、 generic、 inner、 operator、 outer、 rest、 var 、 goto 、const

字面量

字面量：数据在程序中的书写格式

字面量类型 说明 举例
整数类型 不带小数点的数字 666 , -55
小数类型 带小数的数字 3.14 , 4.0
字符串类型 用双引号引起来的内容 “Hello world” , “你好”
字符类型 用单引号引起了的内容，内容只能有一个 ‘A’ , ‘0’ , ‘我’
布尔类型 布尔值，表示真假 true，false
空类型 一个特殊值，空值 null

特殊字符类型字面量：\t \n \r ······

  • \t 制表符：在打印时，把前面字符串的长度补齐到8或8的倍数（最少补1个空格，最多补7个空格）
  • \r 回车符
  • \n 换行符
  • \\：相当于\
  • \'：相当于'
  • \"：相当于"

标识符

标识符：对各种变量、方法和类命名时的字符序列

命名规则：

  • 由数字、字母、下划线(_)和美元符号($)组成
  • 不能以数字开头
  • 区分大小写
  • 不是关键字
  • 不包含空格

小驼峰命名法：

  • 标识符是一个单词时全部小写 （例：name）
  • 标识符由多个单词组成时，第一个单词首字母小写，其他单词首字母大写 （例：firstName）
