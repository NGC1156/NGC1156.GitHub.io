Java-运算符

发表于2022-09-18|更新于2022-09-21|Java-从入门到入土
|阅读量:

运算符介绍

运算符是一种特殊的符号，用以表示数据的运算、赋值和比较等

分类：

  • 算数运算符
  • 关系运算符（比较运算符）
  • 逻辑运算符
  • 赋值运算符
  • 三元运算符
  • 位运算符

算术运算符

对数值型变量进行计算

运算符 运算 范例 结果
+ 加 1 + 1 2
- 减 4 - 2 2
* 乘 2 * 2 4
/ 除 2 / 2 1
% 取模（取余） 11 % 9 2
++ 自增（前）：先运算后赋值 a = 2；b = ++a a = 3；b =3
++ 自增（后）：先赋值后运算 a = 2；b = a++ a = 3；b =2
— 自减（前）：先运算后赋值 a = 2；b = —a a = 1；b =1
— 自减（后）：先赋值后运算 a = 2；b = a— a = 1；b =2
+ 字符串相加 “rubb” + “ish” “rubbish”

除法运算

  1. 整数除以整数，结果为整数（保留整数部分，丢弃小数部分）

  2. 0不能做除数，0.0可以做除数

取模运算

本质：a % b = a - a / b * b

单目运算符（自增运算）

独立使用：

i++等价于i = i + 1

++i等价于i = i + 1

作为表达式使用：

j = ++i等价于：

  1. i = i + 1

  2. j = i

j = i++等价于：

  1. j = i
  2. i = i + 1

关系运算符

结果都是boolean类型

运算符 运算 范例 结果
== 相等于 1 == 2 false
!= 不等于 1 != 2 true
< 小于
> 大于
<= 小于等于
>= 大于等于
instanceof 检查是否是类的对象 “hello” instanceof String true

逻辑运算符

用于连接多个条件（多个关系表达式），结果是boolean值

与

  1. 短路与（&&）：如果第一个条件为 false，则第二个条件不会判断，最终结果为 false，效率高

  2. 逻辑与（&）：不管第一个条件是否为 false，第二个条件都要判断，效率低

或

  1. 短路或（||）：如果第一个条件为 true，则第二个条件不会判断，最终结果为 true，效率高

  2. 逻辑或（|）：不管第一个条件是否为 true，第二个条件都要判断，效率低

非

  • 语法：!(条件)

  • 如果条件本身成立，结果为 false，否则为 true

逻辑异或

  • a^b叫逻辑异或，当 a 和 b 不同时，则结果为 true, 否则为 false

*= a = b 等价于 a =a b
/= a /= b 等价于 a =a / b
%= a %= b 等价于 a =a % b

三元运算符

语法：条件表达式? 表达式 1: 表达式 2;

运算规则：

  • 如果条件表达式为 true，运算后的结果是表达式 1；
  • 如果条件表达式为 false，运算后的结果是表达式 2；

位运算符

进制

  1. 二进制：满2进1，以0b或0B开头表示（0，1）
  2. 十进制：满10进1（0~9）
  3. 八进制：满8进1，以数字0开头表示（0~8）
  4. 十六进制：满16进1，以0x或0X开头表示（0~9、 A~F）

进制转换

  1. 二进制转十进制

    从最低位开始，将各个位上的数提出来，乘以2的位数-1次方，然后求和

  2. 八进制转十进制

    从最低位开始，将各个位上的数提出来，乘以8的位数-1次方，然后求和

  3. 十六进制转十进制

    从最低位开始，将各个位上的数提出来，乘以16的位数-1次方，然后求和

  1. 十进制转二进制

    将该数不断除以2，直到商为0为止，然后将每步得到的余数倒过来，得到对应二进制

  2. 八进制转二进制

    将该数不断除以8，直到商为0为止，然后将每步得到的余数倒过来，得到对应八进制

  3. 十六进制转二进制

    将该数不断除以16，直到商为0为止，然后将每部得到的余数倒过来，得到对应十六进制

  1. 二进制转八进制

​ 从低位开始，将二进制数每三位一组，转成对应的八进制数即可

  1. 二进制转十六进制

    从低位开始，将二进制数每四位一组，转成对应的十六进制数即可

  1. 八进制转二进制

    将八进制数每一位，转成对应的一个三位的二进制数即可

  2. 十六进制转二进制

    将十六进制数每一位，转成对应的一个四位的二进制数即可

原码、反码、补码

  1. 二进制的最高位是符号位:0表示正数, 1表示负数

  2. 正数的原码、反码、补码都一样（三码合一）

  3. 负数的反码 = 它的原码符号位不变，其它位取反（0->1，1->0）

  4. 负数的补码 = 它的反码 + 1

    负数的反码 = 它的补码 - 1

  5. 0的反码，补码都是0

  6. java没有无符号数，换言之，java中的数都是有符号的

  7. 在计算机运算的时候，都是以补码的方式来运算的

  8. 当我们看运算结果的时候，要看他的原码

位运算符

按位与（&）：两位全为1，结果为1，否则为0

按位或（|）：两位有一个为1，结果为1，否则为0

按位异或（^）：两位一个为1，一个为0，结果为1，否则为0

按位取反（~）：0变1，1变0、

算数右移（>>）：低位溢出，符号位不变，并用符号位补溢出的高位

算数左移（<<）：符号位不变，低位补0

逻辑（无符号）右移（>>>）：低位溢出，高位补0

优先级

  1. ()、{} 等
  2. 单目运算符 ++ --
  3. 算术运算符
  4. 位移运算符
  5. 比较运算符
  6. 逻辑运算符（逻辑>短路）
  7. 三元运算符
  8. 赋值运算符
