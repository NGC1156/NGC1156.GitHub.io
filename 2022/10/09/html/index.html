媒体标签

图片标签（img）

属性 说明
src 图片路径
alt 替换文本（图片加载不出来时显示）
title 标题文本（鼠标悬停时显示）
width 宽
height 高

音频标签（audio）

属性名 功能
src 音频路径
controls 显示播放的控件
autoplay 自动播放（部分浏览器不支持）
loop 循环播放

视频标签（video）

属性名 功能
src 视频路径
controls 显示播放的控件
autoplay 自动播放（谷歌浏览器需配合muted实现静音播放）
loop 循环播放

链接标签

属性名 功能
controls 显示播放的控件
target _self（默认，当前窗口跳转，覆盖原网页）
target _blank（新窗口跳转，保留原网页）

表格标签

表格的基本标签

嵌套关系table > tr > td

  • table（表格整体）：包囊tr

  • tr（表格每行）：包囊td

  • td（表格单元格）：包囊内容

表格属性

  • border：边框宽度
  • width：表格宽度
  • height：表格高度

表格标题标签

  • caption（表格大标题）：默认在表格整体顶部居中位置显示

表格单元格标签

  • th（表头单元格）：用于表格第一行，默认内部文字加粗并居中显示

    th标签书写在tr标签内部（用于替换td标签）

结构标签

  • thead：表格头部
  • tbody：表格主体
  • tfoot：表格底部

合并单元格属性

  • rowspan（跨行合并上下合并）：只保留最上的，删除其他

  • colspan（跨列合并左右合并）：只保留最左的，删除其他

表单标签

input标签

type属性值 说明
text 文本框，用于输入单行文本
password 密码框，用于输入密码
radio 单选框，用于多选一
checkbox 多选框，用于多选多
file 文件选择，用于上传文件
submit 提交按钮，用于提交
reset 重置按钮，用于重置
button 普通按钮，默认无功能，后续可配合js添加功能
  • placeholder（占位符）：提示字，配合text和password使用，提示输入内容
  • name（分组）：配合radio单选框使用
  • checked（默认选中）：配合radio和checkbox`使用，默认勾选
  • multiple（多文件选择）：配合file使用，提交多个文件
  • form（表单域标签）：配合按钮使用，提交/重置表单域内内容。action属性：数据地址（提交到……）
  • value：配合按钮使用，给按钮添加提示字

button标签

select下拉菜单

  • select：下拉菜单整体（包裹option）
  • option：下拉菜单的每一项
  • selected属性：默认选中

textarea文本域

  • rows：文本域内可见行数
  • cols：文本域内可见宽度

label标签

  1. 用label标签把内容（文本）包裹
  2. 表单标签添加id属性
  3. label标签添加for属性，并和表单标签id属性相同
  1. 用label标签把表单标签和内容（文本）一起包裹
  2. 删除label标签for属性

语义化标签

无语义的布局标签

  • div标签：一行只显示一个（独占一行）

  • span标签：一行可显示多个

有语义的布局标签

标签名 语义
header 网页头部
nav 网页导航
footer 网页底部
aside 网页侧边栏
section 网页区块
article 网页文章
